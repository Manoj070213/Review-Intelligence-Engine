# Dataset

This project uses the **Datafiniti "Consumer Reviews of Amazon Products"** dataset.

The full source CSV (`amazon_reviews_real.csv`, ~254MB) is **not included in this zip**
to keep it lightweight. Download it free from Kaggle:
https://www.kaggle.com/datasets/datafiniti/consumer-reviews-of-amazon-products

Use the file named `Datafiniti_Amazon_Consumer_Reviews_of_Amazon_Products_May19.csv`,
rename it to `amazon_reviews_real.csv`, and place it in this `data/` folder.

The pipeline's already-processed sample (`raw_reviews.csv`, 1,500 rows) IS included,
so you can run steps 02–04 immediately without downloading anything.
